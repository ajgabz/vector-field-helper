Vector Field Helper

Let V be a vector field defined in spherical coordinates (R, θ, and φ). Let VR, Vθ, and Vφ denote the R, θ, and φ vector components of V, respectively. If a vector component is equal to zero or some other scalar constant (e.g., VR=0), then it is a function of no variables.

Zero Terms
Which vector components is V defined for?
VR=0 Vθ=0 Vφ=0
The component VR is a function of which (if any) variables:
∂VR ∂R =0 ∂VR ∂θ =0 ∂VR ∂φ =0
The component Vθ is a function of which (if any) variables:
∂Vθ ∂R =0 ∂Vθ ∂θ =0 ∂Vθ ∂φ =0
The component Vφ is a function of which (if any) variables: ∂Vφ ∂R =0 ∂Vφ ∂θ =0 ∂Vφ ∂φ =0
