Vector Field Helper

Let V be a vector field defined in cylindrical coordinates (r, φ, and z). Let Vr, Vφ, and Vz denote the r, φ, and z vector components of V, respectively. If a vector component is equal to zero or some other scalar constant (e.g., Vr=0), then it is a function of no variables.

Zero Terms
Which vector components is V defined for?
Vr=0 Vφ=0 Vz=0
The component Vr is a function of which (if any) variables:
∂Vr ∂r =0 ∂Vr ∂φ =0 ∂Vr ∂z =0
The component Vφ is a function of which (if any) variables:
∂Vφ ∂r =0 ∂Vφ ∂φ =0 ∂Vφ ∂z =0
The component Vz is a function of which (if any) variables: ∂Vz ∂r =0 ∂Vz ∂φ =0 ∂Vz ∂z =0
